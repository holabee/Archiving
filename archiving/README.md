# Sanity Blogging 
- Sanity 를 블로그 탬플릿을 이용하여 블로그 생성

<br>

## :fire: 1. 설치 방법 및 실행

```bash

# sanity 패키지 전역 설치
yarn add @sanity/cli -g


# sanity 로그인
sanity login


# sanity 프로젝트 시작
sanity init
sanity start

# sanity 배포
sanity deploy

```
